About this form... The implemantation (see the source code) of this is a form element. It contains 2 radio buttons. The first one is already checked. The submit button is used to submit the form. When we press it the form is submitted with the method specified in the form's method attribute. This is an HTTP verb (GET, PUT, POST, ...) and the form (the data of the form) is submitted in the address specified on the form's action attribute.

Note: The form is data... Actually form elements where included to HTML to enable us to submit data.
